Hi! I'm Xiangyu Wang, a software engineer specializing in vector databases and large-scale vector search.

I currently lead vector search at Ant Group. My work includes building an in-house disk-based vector database designed for hundreds of billions of vectors and used by multiple large-scale internal businesses. I also lead VSAG, an open-source vector search library used by commercial database products including OceanBase, Hologres, and PolarDB.

After graduating from college, I joined Zilliz as #3 employee. I built a GPU-accelerated database prototype that helped the company raise the angel funding of $2.5 million. I later became a core developer and maintainer of Milvus, and helped it become the most popular open-source vector database in the world.

The first time I used a computer I was about 6 years old. The only thing I did on the computer was play the game Crazy Arcade with my cousin. I started learning programming in high school and my first programming language is 易语言(Easy Programming Language) which is a non-English-based programming language. I made some Windows desktop applications with it. EPL is cool! I love it! I learned C/C++ in college and use it as my main programming language for years. 

I like football very much. I am a fan of Cristiano Ronaldo and Robert Lewandowski. And I also play football with my friends after getting off work, about once a week. Shanghai is nice in this regard, there are a lot of places to play football.
